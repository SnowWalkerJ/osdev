# Memory Layout

```
+-----------------------------+
|        kernel space         |
|                             |
|                             |
|                             |
|                             |
|                             |
|                             |
|                             | 0xC0000000 / 0xFFFF800000000000 +
+-----------------------------+
|  non-canonical addresses    | 0xFFFF800000000000 - 1  ~
|                             |
| for amd64 systems impl.48b  | 0x00007FFFFFFFFFFF + 1
+-----------------------------+
|         user space          | 0xC0000000 - 1 / 0x00007FFFFFFFFFFF ~
|                             | 0x00000000     / 0x0000000000000000
|                             |
|                             |
|                             |
|                             |
|                             |
```
